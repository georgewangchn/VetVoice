## 3. 性能测试计划

### 3.1 性能指标 1: 启动时间

**目标指标:** 应用启动时间 < 3 秒

**测试方法:**
- 重复启动应用 10 次
- 记录从双击到主窗口显示的完整时间
- 计算平均启动时间

**测试工具:** 手动计时 / `time` 命令

**测试场景:**
- 冷启动（首次启动）
- 热启动（关闭后立即启动）
- 带参数启动（加载资源路径）

**测试环境:** macOS 14.0 (M3)

### 3.2 性能指标 2: ASR 响应时间

**目标指标:** 语音识别延迟 < 1 秒

**测试方法:**
- 播放标准测试音频
- 记录从录音到识别结果显示的时间差
- 测试不同长度的音频（5s, 10s, 30s）

**测试工具:** 手动计时 / 日志时间戳分析

**测试场景:**
- 短音频识别 (< 5s)
- 中等音频识别 (5-30s)
- 长音频识别 (> 30s)
- 实时识别延迟

**测试环境:** macOS 14.0 (M3), macOS 11.0 (M1)

### 3.3 性能指标 3: LLM 调用时间

**目标指标:** LLM 响应时间 < 20 秒

**测试方法:**
- 调用 LLM API 进行辅助诊断
- 记录从提交请求到收到响应的时间
- 测试不同复杂度的查询

**测试工具:** 日志时间戳分析 / pytest-benchmark

**测试场景:**
- 简单查询（一句话）
- 复杂查询（长对话）
- 并发调用（多个请求）
- API 超时测试

**测试环境:** 本地 vllm 服务 (http://192.168.1.59:8888/v1)

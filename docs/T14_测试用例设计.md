## 2. 正常用例（26个）

### 2.1 功能：病例管理

#### TC001: 创建新病例（填写所有字段）
**用例 ID:** TC001
**优先级:** P1
**前置条件:** 已登录系统
**测试步骤:**
1. 点击"新建病例"按钮
2. 填写所有必需字段（宠物名称、品种、年龄、性别、主人姓名、联系方式）
3. 填写所有可选字段（既往病史、过敏史、备注）
4. 点击"保存"按钮
**预期结果:** 病例创建成功，数据完整保存到数据库

#### TC002: 编辑病例（修改部分字段）
**用例 ID:** TC002
**优先级:** P1
**前置条件:** 已创建病例
**测试步骤:**
1. 选择要编辑的病例
2. 修改部分字段（如年龄、主诉）
3. 点击"保存"按钮
**预期结果:** 病例更新成功，修改的字段正确保存

#### TC003: 删除病例
**用例 ID:** TC003
**优先级:** P1
**前置条件:** 已创建病例
**测试步骤:**
1. 选择要删除的病例
2. 点击"删除"按钮
3. 确认删除
**预期结果:** 病例成功删除，数据库中不再存在

#### TC004: 查询历史病例
**用例 ID:** TC004
**优先级:** P1
**前置条件:** 已创建多个病例
**测试步骤:**
1. 在搜索框输入查询条件（如宠物名称、主人姓名、日期范围）
2. 点击"搜索"按钮
**预期结果:** 显示符合查询条件的病例列表

#### TC005: 复制病例数据
**用例 ID:** TC005
**优先级:** P1
**前置条件:** 已创建病例
**测试步骤:**
1. 选择要复制的病例
2. 点击"复制"按钮
3. 修改部分字段
4. 保存为新病例
**预期结果:** 病例数据成功复制，可独立编辑
